Telegram News Bot
Этот проект представляет собой Telegram-бота, который собирает новости из Google и Яндекс, используя парсинг, и отправляет их пользователю. Бот также включает функциональность для обработки капчи с помощью сервиса RuCaptcha.

Структура проекта
bot_main.py: Основной файл для запуска Telegram-бота.

config.py: Файл для хранения API-токена бота.

google_news.py: Скрипт для парсинга новостей из Google.

yandex_news.py: Скрипт для парсинга новостей из Яндекс.

use_news.py: Скрипт, который объединяет новости из разных источников.

testyandex.py: Скрипт для тестирования обработки капчи Яндекс.

Установка и запуск
Клонируйте репозиторий:

Copy

cd ваш-репозиторий
Установите необходимые зависимости:

bash
Copy
pip install -r requirements.txt
Создайте файл config.py и добавьте в него API-токен вашего бота:

python
Copy
API_TOKEN = 'ваш-токен'
Запустите бота:

bash
Copy
python bot_main.py
Использование
После запуска бота, вы можете взаимодействовать с ним через Telegram. Бот будет отправлять вам новости из Google и Яндекс.

Обработка капчи
Для обработки капчи используется сервис RuCaptcha. Убедитесь, что у вас есть аккаунт и API-ключ от RuCaptcha, если вам нужно обрабатывать капчи.
